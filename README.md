## 실행 환경
Ubuntu20.04  
Ros2-foxy  
Gazebo11  
rviz2  

## 명령어
폴더를 만들기 ccpp-robot/ccpp_ws  colcon build  
cd src  
git clone https://github.com/starloving/ccpp-robot.git  
cd ..  
colcon build  
ccpp_ws에 source install/setup.bash  
ros2 launch robot_ccpp robot.launch.py
